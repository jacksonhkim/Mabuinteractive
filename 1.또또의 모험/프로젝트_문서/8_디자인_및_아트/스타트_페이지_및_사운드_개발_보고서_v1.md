## 1. 스타트 페이지 비주얼 명세 (Visual Specs)

대표님의 지시 사항을 바탕으로 고화력 비주얼과 브랜드 정체성을 결합한 시작 화면을 구축했습니다.

### 1.1 배경 화면 (Background)
- **에셋 명**: `bg_start.png`
- **구현 방식**: 
    - CSS `#start-screen` 배경 이미지로 고해상도 출력 (`background-size: cover`).
    - JS 렌더링 엔진 내 캔버스 드로잉 로직 병행 (로딩 지연 방지).

### 1.2 타이틀 디자인 (Title Logo)
- **텍스트**: `또또의 모험 (The Adventure of Toto)`
- **폰트**: **여기어때 잘난체 (Yeogioette Jallanche)** - 둥글고 볼륨감 있는 Retro-Modern 감성.
- **색상 (Honey Palette)**: 
    - `#FFFB00` (상단) → `#FFD54F` (중앙) → `#FFA000` (하단) 그라데이션 적용.
- **3D 입체 효과**:
    - **다중 섀도우**: 또또의 줄무늬 색상(#3E2723)을 활용한 4단계 레이어 그림자.
    - **원근감**: `perspective(500px) rotateX(10deg)`를 통한 공간감 부여.
- **배치**: 배경 이미지의 **무지개 곡선 위치**에 정밀 배치 (`margin-top: 150px`).

### 1.3 UI 및 하단 크레딧
- **시작 메시지**: 중앙 또또 캐릭터를 가리지 않도록 하단 배치 (`margin-top: 280px`).

## 2. 사운드 시스템 R&D 명세 (Sound Engine)

절차적 음향 합성(Procedural Audio Synthesis) 기술을 도입하여 상황별 몰입감을 극대화했습니다.

### 2.1 BGM 상황별 테마 (State-based BGM)
| 구간 | 테마 성격 | 기술적 특징 |
| :--- | :--- | :--- |
| **START** | 모험의 시작 (웅장/평화) | C Major 스케일, Triangle 파형, 도트 감성 멜로디 |
| **SELECT** | 캐릭터 선택 (경쾌/발랄) | 빠른 템포, Square 파형, 스타카토 아르페지오 |
| **STAGE 1-3** | 안정기 (숲의 평화) | 높은 음역대 멜로디, Reverb 효과로 공간감 확장 |
| **STAGE 4-7** | 긴장기 (위기와 성취) | Minor 스케일 전환, Sawtooth 파형의 날카로운 베이스 |
| **STAGE 8-10** | 고조기 (최종 결전) | 최고 속도 템포, 복합 레이어링 화음 |
| **ENDING** | 결실 (승리와 여운) | Sine 파형의 부드러운 선율, 긴 Decay 값 적용 |

### 2.2 기술적 구현 (Mabu Sound Engine)
- **엔진**: Web Audio API 기반 실시간 신시사이저 소리 생성.
- **최적화**: 외부 오디오 파일 로드 없는 0KB 사운드 시스템 구축.
- **인터랙션**: 사용자 첫 클릭 시 오디오 컨텍스트 자동 활성화 및 BGM 전환.
