# 또또의 모험 사운드 기획 및 리소스 명세서

## 1. 사운드 컨셉 개요
- **장르:** 뉴트로 픽셀 슈팅
- **방향성:** 8비트 레트로 감성과 현대적 오케스트레이션이 결합된 '하이브리드 픽셀 사운드'. 꿀벌 또또의 경쾌함과 기계 제국의 묵직한 기계음을 대조시킴.
- **최적화 전략:** Web Audio API를 활용한 오디오 스프라이트 적용, 모든 파일은 `.webp` 또는 `.ogg` (96kbps) 포맷으로 경량화.

## 2. 배경음악 (BGM) 리스트 (총 10개 스테이지)

| 스테이지 | 테마 | 감정/분위기 | 사운드 구성 특징 |
| :--- | :--- | :--- | :--- |
| **Stage 1** | 평화로운 꽃밭 | 희망적, 경쾌함 | 하이햇 위주의 빠른 비트, 밝은 신스 멜로디 |
| **Stage 3** | 은빛 폭포 | 웅장함, 시원함 | 물소리 환경음 믹싱, 리버브가 강조된 사운드 |
| **Stage 6** | 불타는 고향 | 긴박함, 비극 | 긴박한 스트링 앙상블, 화염 타는 듯한 디스토션 |
| **Stage 10** | 황금의 심판 | 비장함, 절정 | 풀 오케스트라 + 8비트 베이스라인의 결합 |

## 3. 효과음 (SFX) 설계 (우선순위 기반)

| SFX 명칭 | 발생 조건 | 사운드 특징 (청각적 피드백) | 우선순위 |
| :--- | :--- | :--- | :--- |
| **Stinger Shot** | 플레이어 발사 | '푝!' 하는 경쾌하고 날카로운 고주파음 | High |
| **Enemy Explode** | 적 파괴 | '콰콰쾅!' 묵직한 저음역대 중심의 타격음 | High |
| **Bee Wing** | 플레이어 이동 | '위잉-' 하는 미세한 고주파 루프 사운드 | Low |
| **Collect Item** | 아이템 획득 | '띠링!' 하는 맑고 청아한 상향식 멜로디 | Medium |
| **Hit Damage** | 플레이어 피격 | '윽!' 하는 짧고 둔탁한 노이즈 사운드 | Very High |

## 4. 기술적 가이드라인
- **Audio Sprite:** 모든 SFX를 하나의 파일에 합쳐 로딩 횟수 단축 (0.1s 간격 배치).
- **Dynamic Pitch:** 탄환 발사 시 피치(Pitch)를 ±5% 랜덤화하여 청각적 피로도 감소.
- **Occlusion:** 기계 중심 스테이지에서는 저역 통과 필터(LPF)를 사용하여 공간감 부여.
