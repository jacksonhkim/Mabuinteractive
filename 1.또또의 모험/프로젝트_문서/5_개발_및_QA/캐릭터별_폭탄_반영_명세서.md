# 🛠️ 캐릭터별 폭탄(궁극기) 비주얼 반영 명세서
**작성자**: 또또 (Tech Lead)
**검토**: 전사 에이전트 (Dev Team)
**최종 반영일**: 2026-02-12

본 문서는 대표님 지시사항에 따라 반영된 캐릭터별 고유 폭탄 디자인의 기술적 명세와 시각적 가이드라인을 정의합니다.

## 1. 구현 레이어 및 파일 구조
| 항목 | 파일 경로 | 주요 역할 |
| :--- | :--- | :--- |
| **Logic** | `entities.js` | `useBomb()` 시점에서 `charId`를 바인딩하여 폭탄 데이터 객체 생성 |
| **Drawing** | `pixel_art_v2.js` | `drawPixelBombEffectV2()`에서 ID별 분기 렌더링 및 헬퍼 함수(Star, Heart 등) 정의 |
| **Rendering** | `renderer.js` | 매 프레임 업데이트되는 폭탄 리스트를 순회하며 `charId` 값을 파라미터로 전달 |

## 2. 캐릭터별 시각적 스펙 (Implemented)

### 🐧 TOTO (Honey Bee)
- **Primary Color**: #FFD600 (Honey Gold)
- **Shape**: Honeycomb Hexagon Pattern
- **Logic**: 중심점에서 외곽으로 6방향 벌집 문양 생성

### 🧚 LULU (Fairy)
- **Primary Color**: #F06292 (Magic Pink)
- **Shape**: Sparkle Stars
- **Logic**: 무작위 위치에 크기가 다른 별 문양 12개 이상 생성하여 몽환적 연출

### 🐘 KAKA (Elephant Warrior)
- **Primary Color**: #8D6E63 (Dusty Brown)
- **Shape**: Rock Fragments
- **Logic**: 불규칙한 사각형 파편과 굵은 입자를 사방으로 비산시켜 파괴력 시각화

### 🐈 MOMO (Love Cat)
- **Primary Color**: #EF5350 (Scarlet Red)
- **Shape**: Radiant Hearts
- **Logic**: 베지어 곡선으로 그려진 하트 문양 8개를 동심원 형태로 확장

### 🐰 PIPI (Rabbit Speedster)
- **Primary Color**: #00BCD4 (Cyan/Sonic)
- **Shape**: Sonic Rings / Shockwaves
- **Logic**: 2중/3중의 원형 파동(Ring)을 확산시켜 음속 타격 연출

## 3. QA 및 성능 데이터
- **Frame Rate**: 폭발 연출 시에도 60FPS 안정적 유지 확인 (Particle Culling 적용)
- **Collision**: 시각적 반지름과 실제 데미지 판정 반지름(800px)의 동기화 완료
