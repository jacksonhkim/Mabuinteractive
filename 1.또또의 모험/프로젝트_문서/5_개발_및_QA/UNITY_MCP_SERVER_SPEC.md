# 🛠 [Mabu Interactive] Unity MCP Server Specification

## 📌 개요
유니티를 이용한 게임 개발 시 협업 및 기능 확장을 위해 구축된 MCP(Model Context Protocol) 서버 정보입니다.

## 📡 서버 인프라 정보
- **서버 명칭:** Unity MCP Server (Local Integration)
- **접속 포트:** `8080`
- **구동 상태:** Active (User Started)
- **주요 용도:** 
  - 유니티 프로젝트와 AI 어시스턴트(피터, 또또, 보) 간의 실시간 데이터 연동
  - 유니티 API 및 에디터 기능 자동화
  - 차기 유니티 기반 프로젝트 개발 시 기술적 교두보 역할

## 🚀 개발팀 안내 사항
- **또또(Dev Lead):** 향후 유니티 프로젝트 전환 시 해당 서버를 통해 에셋 관리 및 빌드 자동화 스크립트를 연결할 것.
- **보(Planning Lead):** 유니티 엔진의 연산 결과나 게임 상태 데이터를 AI가 실식간으로 분석하여 기획 보조할 수 있도록 활용 계획 수립 요망.

## 📂 관련 설정
- MCP 설정 파일(`mcp-config.json` 등)에서 해당 포트(8080)를 우선적으로 참조하도록 구성하십시오.
